# humanizer-ru

> [English version](README.md)

Плагин для Claude Code и Cowork. Убирает признаки AI-генерации из русскоязычного текста.

Английский [humanizer](https://github.com/blader/humanizer) для русского не работает: маркеры AI-текста в двух языках разные. Em-dash overuse и copula avoidance мало что значат, когда проблема в канцелярите, кальках с английского и отсутствии частиц "же", "ведь", "вот".

## Что внутри

37 паттернов AI-генерации в 8 категориях: контентные (пустые открытия, размытые авторитеты, раздувание значимости), языковые (канцелярит, кальки, злоупотребление "является", однообразие синтаксиса), стилистические (тире, болд, кавычки с адаптацией под контекст), коммуникативные (артефакты чатбота, заискивание, водянистость), морфологические (падежи, деепричастия), тональные (стерильность, отсутствие частиц и иронии), структурные (разорванные абзацы, галлюцинации), человеческие мелочи (идеальная грамотность и вылизанная типографика как маркеры AI).

Плюс 5 формул статей: вовлекающая, экспертная, продающая, новостная, сторителлинг.

Калибровка голоса: можно дать образцы своего текста, скилл подстроится под стиль.

## Установка

Как плагин (рекомендуется):

```
/plugin install humanizer-ru
```

Или добавь репозиторий как маркетплейс:

```
/plugin marketplace add ilyautov/humanizer-ru
/plugin install humanizer-ru@ilyautov-plugins
```

Как скилл вручную:

```bash
mkdir -p ~/.claude/skills/humanizer-ru
curl -o ~/.claude/skills/humanizer-ru/SKILL.md \
  https://raw.githubusercontent.com/ilyautov/humanizer-ru/main/skills/humanizer-ru/SKILL.md
```

## Использование

Просто попроси Claude очеловечить текст:

```
Очеловечь этот текст: [вставить текст]
Перепиши, звучит как робот: [вставить текст]
```

Триггеры: "очеловечь", "убери следы нейросети", "сделай живым", "звучит искусственно", "перепиши как человек".
