# humanizer-ru

> [English version](README.md)

Плагин для Claude Code и Cowork. Убивает запах нейросети в русском тексте.

Английский [humanizer](https://github.com/blader/humanizer) тут не поможет. У русского AI-текста свои маркеры: канцелярит, кальки с английского синтаксиса, пропавшие частицы "же" и "ведь", от которых русская речь дышит. С английскими паттернами это не пересекается вообще.

## Что даёт

44 паттерна в 10 категориях - от канцелярита и калек до эмоциональной стерильности, паттернов убеждения и информационного ритма. Жёсткие баны: конструкции, которые кричат "это писал GPT" (включая длинные тире, которые детекторы теперь считают статистически). Секция о том, как именно работают детекторы (perplexity, burstiness, морфология). Пять формул статей (блог, экспертная, продающая, новостная, сторителлинг). Калибровка голоса, если скормить образцы своего текста.

## Установка

Плагин (рекомендуется):

```
/plugin install humanizer-ru
```

Маркетплейс:

```
/plugin marketplace add ilyautov/humanizer-ru
/plugin install humanizer-ru@ilyautov-plugins
```

Вручную:

```bash
mkdir -p ~/.claude/skills/humanizer-ru
curl -o ~/.claude/skills/humanizer-ru/SKILL.md \
  https://raw.githubusercontent.com/ilyautov/humanizer-ru/main/skills/humanizer-ru/SKILL.md
```

## Режимы

**Полное редактирование** (по умолчанию). Все 44 паттерна, калибровка голоса, тройной аудит. Для текста, который должен звучать по-человечески.

**Аудит**. Только диагностика - список найденных паттернов с приоритетом (A-D). Текст не трогает.
```
Проверь этот текст на AI-маркеры: [вставить текст]
```

**Точечная правка**. Работает только с указанной категорией, остальное не трогает.
```
Убери канцелярит: [вставить текст]
```

## Использование

Попроси Claude по-русски:

```
Очеловечь этот текст: [вставить текст]
Перепиши, звучит как робот: [вставить текст]
```

Триггеры: "очеловечь", "убери следы нейросети", "сделай живым", "звучит искусственно", "перепиши как человек".

## До / После

**Полное редактирование:**

> В современном мире искусственный интеллект играет всё более важную роль в различных сферах деятельности. Стоит отметить, что данная технология является мощным инструментом для оптимизации рабочих процессов.

превращается в:

> За последний год я внедрил AI-инструменты в три проекта. Два ускорились вдвое. Третий развалился, потому что команда перестала проверять то, что выдаёт модель. AI работает, когда понимаешь его ограничения.

Пять жёстких банов сработали в двух предложениях. Типичная картина.

**Аудит** того же текста выдаст:

```
[A] Пустое открытие: "В современном мире..."
[A] HARD BAN: "является мощным инструментом"
[A] HARD BAN: "играет важную роль"
[A] HARD BAN: "Стоит отметить, что..."
[B] Канцелярит: "оптимизации рабочих процессов"
[B] Раздувание: "всё более важную роль в различных сферах"
```
